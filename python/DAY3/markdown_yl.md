## ✅ Ülesanded

### ÜL1. Too andmed API-st
- [ ] Too kasutajate andmed:  
  `https://jsonplaceholder.typicode.com/users`  
- [ ] Too todo andmed:  
  `https://jsonplaceholder.typicode.com/todos`

---

### ÜL2. Töötle andmeid
- [ ] Seo `todos` vastava kasutajaga `userId` abil.  
- [ ] Arvuta iga kasutaja kohta:
  - Kokku ülesandeid  
  - Tehtud ülesandeid  
  - Tegemata ülesandeid
